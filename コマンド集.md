# ビルドコマンド

## ① Windows版インストーラを生成してGitHubに自動公開

PowerShellまたはターミナルで実行👇
```bash
npx electron-builder --win --publish always
```


## Windows でビルド
```bash
npm run dist -- --win --x64
```

## Macでビルド
```bash
npm run dist -- --mac
```
