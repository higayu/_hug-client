# Electronでの状態管理ストアについて

Electronでは状態管理ストアを使用できます。ElectronはNode.jsベースなので、一般的なJavaScript/TypeScriptの状態管理ライブラリが使用可能です。

## 使用可能な状態管理ライブラリ
Redux / Redux Toolkit

標準的な選択肢
ElectronとReactの組み合わせでよく使われる

Zustand
軽量でシンプル
TypeScriptとの相性が良い

Jotai / Recoil
アトミックな状態管理
細かい粒度での更新が可能

Valtio
プロキシベースの状態管理
ミュータブルな操作感

Context API + useReducer
React標準の機能
追加ライブラリ不要
推奨が適用されるケース

現在のAppStateContextで十分な場合: そのまま継続可能
より複雑な状態管理が必要な場合: ZustandやRedux Toolkitを検討
パフォーマンス最適化が必要な場合: JotaiやRecoilを検討
現在のプロジェクトではAppStateContextを使用していますが、必要に応じて上記のライブラリに移行できます。
